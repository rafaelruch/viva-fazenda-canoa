## Features

- **Hero com vídeo fullscreen** (mashup 20s de 3 vídeos, autoplay/loop/muted)
- **Copy emocional** do PDF `analise LP Fazenda Canoas.pdf`
- **Cormorant Garamond italic** para headlines + Manrope para body
- **CTA corten `#A7522C`** (terceira cor elegante)
- **Formulário na metade da página** (alto ticket = gerar desejo antes)
- **Modal escuro** com floating labels e select dark
- **CountUp animado** (608k m², 2000m orla, 34% valorização, 22 amenidades)
- **14 seções** fiéis ao PDF de análise
- **SEO completo** (JSON-LD, meta tags, OG, performance, security headers)
- **Sem fuga externa** — toda conversão é interna (formulário + modal)

## Instalação

1. Plugin: Admin → Plugins → Ativar "Fazenda Canoa — Opções & Leads"
2. Tema: Admin → Aparência → Temas → Ativar "Viva Fazenda Canoa"
3. Configurações → Fazenda Canoa → Revisar contatos
